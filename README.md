# Customer Churn Prediction Model

An end-to-end machine learning project predicting telecom customer
churn using XGBoost, deployed as an interactive Streamlit web app.

## Tech Stack
Python · Pandas · XGBoost · Scikit-Learn · Streamlit · Seaborn

## Project Stages
- [x] Stage 1: Project setup and dataset
- [ ] Stage 2: Exploratory Data Analysis
- [ ] Stage 3: Data cleaning and feature engineering
- [ ] Stage 4: XGBoost model building and evaluation
- [ ] Stage 5: Streamlit web app deployment

## Business Impact
Identifying high-risk customers before they leave can save
an estimated ₹12L+ in annual revenue.

## Dataset
Telco Customer Churn — 7,043 customers, 21 features — Kaggle
